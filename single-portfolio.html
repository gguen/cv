Outil de simulation

Description

Création et réalisation d'un outil web de simulation du champ direct des enceintes Ray-On

L'utilisateur peut choisir le modèle d'enceinte, la hauteur de pose, la puissance. Une fois la configuration finalisée, il est possible d'ajouter des marqueurs de niveaux sonores sur la carte et d'exporter le résulat sous forme d'image

Date: 2015

HTML, css, javascript, script lua pour compiler l'ensemble dans un seul fichier HTML
